## Not Everything Belongs in Git


## Write Your First .gitignore


```bash
python3 calculator.py
```


```bash
git status
```


```bash
cat << 'EOF' > .gitignore
# Compiled Python files
__pycache__/
*.pyc

# Virtual environment
venv/

# Environment variables (secrets)
.env

# IDE settings
.vscode/
.idea/

# OS files
.DS_Store
Thumbs.db
EOF
```


```bash
git status
```


```bash
git add .gitignore
git commit -m "Add .gitignore file"
```


## Pattern Matching in a Nutshell


```
*.log
!important.log
```


## Already Tracked? Here's the Fix


```bash
git rm --cached <file>
```


```bash
git rm --cached .env
git commit -m "Stop tracking .env file"
```

## Ignore Files Across Every Repo


```bash
git config --global core.excludesfile ~/.gitignore_global
```


```bash
cat << 'EOF' > ~/.gitignore_global
# macOS
.DS_Store

# Windows
Thumbs.db

# Linux
*~

# IDE settings
.vscode/
.idea/
EOF
```
