# Your First Save Point


## Set Up the Project Folder


```bash
mkdir ~/my-calculator && cd ~/my-calculator
```


## Turn a Folder Into a Repo


```bash
git init
```


```bash
Initialized empty Git repository in /home/your-user/my-calculator/.git/
```


## See What Git Sees


```bash
git status
```


```bash
On branch main
No commits yet
nothing to commit
```


## Write Some Actual Code


```bash
cat << 'EOF' > calculator.py
# calculator.py - A simple calculator

def add(a, b):
    """Add two numbers and return the result."""
    return a + b

# Try it out
result = add(5, 3)
print(f"5 + 3 = {result}")
EOF
```


```bash
python3 calculator.py
```

## Change, Stage, Commit - Repeat


```bash
cat << 'EOF' > calculator.py
# calculator.py - A simple calculator

def add(a, b):
    """Add two numbers and return the result."""
    return a + b

def subtract(a, b):
    """Subtract b from a and return the result."""
    return a - b

# Try it out
result_add = add(5, 3)
result_sub = subtract(10, 4)
print(f"5 + 3 = {result_add}")
print(f"10 - 4 = {result_sub}")
EOF
```


```bash
git status
```


```bash
Changes not staged for commit:
  modified: calculator.py
```


```bash
git diff
```


```bash
git add calculator.py
git commit -m "Add subtraction function"
```
